# Developer Portfolio

A personal developer portfolio website built to showcase my projects, technical skills, internship experience, education, and contact information in a clean and professional format.

## Live Website
🔗 [View Portfolio](https://ashna-dev01.github.io/developer-portfolio/)

---

## Overview
This portfolio website was created to present my background as a **B.Sc. Computer Science student** and an **aspiring Python Full Stack Developer**. It includes information about my skills, internships, education, projects, and ways to connect with me.

---

## Features
- Responsive portfolio website design
- Hero section with introduction and career focus
- About section with background and interests
- Skills section for technical abilities
- Education section
- Internship experience section
- Featured project section
- Contact section with GitHub, LinkedIn, and email
- Smooth scrolling navigation
- Responsive layout for desktop, tablet, and mobile screens

---

## Technologies Used
- HTML5
- CSS3
- Responsive CSS / Media Queries
- Git
- GitHub Pages
